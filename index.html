Sydney Feldman and Gracie Dunaway

Project Website

Project Dataset¶

The dataset we have decided to use is called US School Scores, which was found on Kaggle. This data set has 99 columns of information regarding SAT scores by state and US territory from 2005-2015. Although all the information in this dataset is based on when the SAT was out of 2400 points instead of the current 1600 points, it is still relevant to our goals. The only difference between the two tests is that the written portion became optional but the data set we have does not include the writing section so the score is out of 1600 still.

The columns cover a range of topics including scores from different sections of the SAT, subject specific GPA, the average scores based on family income, and many more data points.

You can find the dataset used here.

Project Goals¶

Our goal of this project is to discover different factors that affect student SAT scores. Through our first analysis, shown below, we were able to test hypotheses and different questions we had. From the data we gathered, we would like to focus on the differences between regions and states, and see if there are any interesting trends. Another piece of data we are considering using in the future is the political stances of each state. Our main dataset compared to this additional one can show us if there is a correlation betweens scores and political affiliation, and more generally politics effect on education.

Table #1¶

One of our hypothesis is that family income has a positive correlation with SAT scores, meaning the higher the family income, the higher the average SAT score. The questions we have based on this hypothesis are if there is a difference between math and english scores, and if the effects of these socioeconomic factors vary by year and state.

The first table we have created below includes data about SAT scores related to family income. We have added also added columns to show the percentage of test-takers in each family income group of the total test-takers for each row. We created a new column for total test takers since the total number of test takers do not correspond to the total number of test takers recorded based on income. To do this, we will create a new table, using columns from the main dataframe.

We tidied this data by removing the state abbreviation column, as it is redundant here, in addition to changing column names to better reflect the data it hold.
